Data Science Process: Explore and Clean Data

This step is for the data source we chose to build a model on
  • Step 1: Load the data
    • Load the data source files
    • Merge the data if needed
  • Step 2: Understand the data
    • Explore the data
    • Understand the predictors, what they mean in real life
    • Understand the values of each predictors
  • Step 3: Handle missing data
    Are there any missing values, if there are:
    • Can we impute them based on some algorithm
    • Remove or ignore them
    • Assume values based on common sense or prior knowledge
  • Step 4: Identify Epilepsy(response variable) Records
    In our data a patient is assumed to be epileptic is one or more conditions are satisified in the dataset. We need to check all the conditions in the data and determine if the patient is epileptic.
  • Step 5: Save cleaned data
    Save our cleaned dataset so we can use it later in our modeling step
